Toggle Group

alpha

A single-select button group for switching views, filters, or modes. Active state derives from the color engine via --nudge-l and --nudge-c. Inherits --hue from its context.

API

classes
.toggle-groupContainer. Inline flex, rounded, clipped.
.toggle-itemEach option. Transparent bg, contrast-derived text.
.activeSelected state. Nudge-l: -12, nudge-c: +0.08.
.icon-onlySquare padding, grid centered. For SVG icons.
modifiers
.pill100vw radius. Extra inline padding on ends.
.smOn group. --type: -0.5, tighter padding.
.lgOn group. --type: 0.2, wider padding.
engine knobs
--hueInherited. Sets palette for the group + active state.
--nudge-lGroup uses -2. Active uses -12.
--nudge-cActive uses +0.08 for chromatic pop.
--contrastIdle: 0.5. Hover: 0.75. Active: 1.
--typeBase: -0.2. Sm: -0.5. Lg: 0.2.
a11y
aria-labelRequired on .icon-only items.
roleConsider role="radiogroup" + role="radio" for strict a11y.
reactivity
data-classDatastar. Bind .active to a signal.
data-on-clickDatastar. Set the signal on click.
html
<div data-signals="{view: 'board'}"> <div class="toggle-group surface"> <button class="toggle-item" data-class:active="$view === 'list'" data-on-click="$view = 'list'" >List</button> <button class="toggle-item" data-class:active="$view === 'board'" data-on-click="$view = 'board'" >Board</button> <button class="toggle-item" data-class:active="$view === 'timeline'" data-on-click="$view = 'timeline'" >Timeline</button> </div> </div>
