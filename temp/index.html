Toggle Button

Toggle Group

alpha

A single-select button group for switching views, filters, or modes. Active state derives from the color engine via --nudge-l and --nudge-c. Inherits --hue from its context.

View:  
/* Toggle Button [99% DONE] */
@layer component.complex {
    .toggle-group { display: inline-flex;
        button { --contrast: 0.5;
            &:has(+ button) { border-start-end-radius: 0; border-end-end-radius: 0; }
            & + button      { border-start-start-radius: 0; border-end-start-radius: 0;
                            border-inline-start: 1px solid var(--_border); }
            &:hover { --contrast: 0.75; }
            &[aria-pressed="true"] { --surface:19; --nudge-c:0.2; }
            &[aria-pressed="true"] + button,
            & + button[aria-pressed="true"] { border-inline-start-color: transparent; }
        }
    }
}
