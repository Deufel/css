The selected hue family determines the entire color context for a subtree. Set --hue on a container and every child component — buttons, tags, inputs, cards — derives from it automatically.

Use --nudge-h for relative shifts within a family (e.g., complementary badges). Use --hue directly when you want a completely new palette (e.g., status colors, section themes).
