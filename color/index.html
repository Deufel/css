five design-system palettes, each as a config preset of the same formula
note: these are flavor approximations, not exact reproductions. each card is a set of six config values that evokes the named system's palette character — its warmth, restraint, saturation, neutrality. the real systems use much more (typography, spacing, components, motion). this demo isolates one variable — the color palette — to show how much variety lives in six numbers.
